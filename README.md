# Greenwheels Kosten Calculator

## Beschrijving (Nederlands)
Dit project is een tool voor het berekenen van de kosten van autohuur bij **GreenWheels**, een autodeelplatform. De tarieven zijn hardcoded in het programma, wat betekent dat eventuele wijzigingen handmatig in de code moeten worden aangepast.

Met deze tool kun je:
- De goedkoopste huurprijs berekenen op basis van verschillende autotypes en abonnementen.
- De kosten voor zowel korte als lange huurperiodes berekenen.
- Opties vergelijken op basis van je huidige abonnement.

Het programma is opgesplitst in twee delen:
1. **Gebruikersinteractie** – Vraagt gebruikersinvoer en toont resultaten.
2. **CostCalculator** – Voert berekeningen uit en retourneert resultaten.

### Belangrijk
De tarieven in het programma zijn gebaseerd op de prijzen van **GreenWheels** en kunnen afwijken van de actuele prijzen. Controleer altijd de meest recente tarieven via de officiële website:  
[GreenWheels Tarieven](https://www.greenwheels.nl/tarieven)

Meer informatie over GreenWheels kun je vinden op hun website:  
[GreenWheels](https://www.greenwheels.nl)

## Description (English)
This project is a tool for calculating car rental costs for **GreenWheels**, a car-sharing platform. The rates are hardcoded in the program, meaning that any changes must be manually updated in the code.

With this tool, you can:
- Calculate the cheapest rental price based on different car types and subscription plans.
- Compute costs for both short-term and long-term rentals.
- Compare options based on your current subscription.

The program is split into two main parts:
1. **User Interaction** – Collects user input and displays results.
2. **CostCalculator** – Performs calculations and returns results.

### Important
The rates in the program are based on **GreenWheels** prices and may differ from the latest rates. Always check the most recent prices via the official website:  
[GreenWheels Rates](https://www.greenwheels.nl/tarieven)

For more information about GreenWheels, visit their website:  
[GreenWheels](https://www.greenwheels.nl)
